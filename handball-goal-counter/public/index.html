Handball Goal Counter

Sistema automatizado de detección de goles desde videos de YouTube

YOLOv8 DeepSORT yt-dlp OpenCV Vercel
POST

/api/analyze-video

📝 Ejemplo de Request:

{
  "youtube_url": "https://www.youtube.com/watch?v=VIDEO_ID"
}

✅ Ejemplo de Respuesta:

{
  "success": true,
  "data": {
    "goals_detected": 7,
    "goal_timestamps": [45.2, 89.7, 134.5, ...],
    "team_scores": {
      "team_a": 4,
      "team_b": 3
    },
    "processing_time": 187.34
  }
}

🚀 Uso con cURL:

curl -X POST https://tu-app.vercel.app/api/analyze-video \
  -H "Content-Type: application/json" \
  -d '{"youtube_url": "https://www.youtube.com/watch?v=..."}'
